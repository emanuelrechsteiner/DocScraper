# Inset
Applies a negative margin to allow content to bleed into the surrounding container.
[View source](https://github.com/radix-ui/themes/blob/main/packages/radix-ui-themes/src/components/inset.tsx)[Report an issue](https://github.com/radix-ui/themes/issues/new?title=\[Inset\] Issue)[View in Playground](https://www.radix-ui.com/themes/playground#inset)
