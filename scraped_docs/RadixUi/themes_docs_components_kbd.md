# Kbd
Represents keyboard input or a hotkey.
[View source](https://github.com/radix-ui/themes/blob/main/packages/radix-ui-themes/src/components/kbd.tsx)[Report an issue](https://github.com/radix-ui/themes/issues/new?title=\[Kbd\] Issue)[View in Playground](https://www.radix-ui.com/themes/playground#kbd)
`Shift + Tab`
```

<Kbd>Shift + Tab</Kbd>


```

## [API Reference](https://www.radix-ui.com/themes/docs/components/kbd#api-reference)
This component is based on the `kbd` element and supports [common margin props](https://www.radix-ui.com/themes/docs/overview/layout#margin-props).
Prop| Type| Default  
---|---|---  
`asChild`Prop description| `boolean`| No default value  
`size`| `Responsive<enum>`See full type| No default value
