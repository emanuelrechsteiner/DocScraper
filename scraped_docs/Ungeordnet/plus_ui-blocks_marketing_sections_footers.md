# Footers
Use these Tailwind CSS footer examples to include useful information at the bottom of each page, like a sitemap to help users navigate your site, links to your social media profiles, or a newsletter sign up form. These examples are designed and built by the Tailwind CSS team, and include a variety of different styles and layouts.
