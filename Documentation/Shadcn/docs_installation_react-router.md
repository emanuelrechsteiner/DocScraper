# React Router
Install and configure shadcn/ui for React Router.
### [](https://ui.shadcn.com/docs/installation/react-router#create-project)Create project
pnpmnpmyarnbun
```
pnpm create react-router@latest my-app

```

Copy
### [](https://ui.shadcn.com/docs/installation/react-router#run-the-cli)Run the CLI
Run the `shadcn` init command to setup your project:
pnpmnpmyarnbun
```
pnpm dlx shadcn@latest init

```

Copy
### [](https://ui.shadcn.com/docs/installation/react-router#add-components)Add Components
You can now start adding components to your project.
pnpmnpmyarnbun
```
pnpm dlx shadcn@latest add button

```

Copy
The command above will add the `Button` component to your project. You can then import it like this:
app/routes/home.tsx
```
import { Button } from "~/components/ui/button"
import type { Route } from "./+types/home"
export function meta({}: Route.MetaArgs) {
 return [
  { title: "New React Router App" },
  { name: "description", content: "Welcome to React Router!" },
 ]
}
export default function Home() {
 return (
  <div className="flex flex-col items-center justify-center min-h-svh">
   <Button>Click me</Button>
  </div>
 )
}
```
